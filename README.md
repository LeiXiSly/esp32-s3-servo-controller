# ESP32-S3 360° 舵机 WiFi 控制器

基于 ESP32-S3 的 360 度连续旋转舵机 WiFi 远程控制器，支持通过 Web 界面实时控制舵机转速和方向。

## 🌟 特性

- ✅ **360° 连续旋转** - 支持正转、反转、停止控制
- ✅ **Web 界面控制** - 手机/电脑浏览器直接访问
- ✅ **无级调速** - -100% 到 100% 精确速度控制
- ✅ **实时反馈** - 2 秒自动刷新状态
- ✅ **可视化动画** - 舵机旋转状态实时显示
- ✅ **预设档位** - 一键切换反转/停止/慢速/快速

## 📦 所需材料

| 物品 | 数量 | 备注 |
|------|------|------|
| ESP32-S3 开发板 | 1 块 | Xiao ESP32S3 或 ESP32S3-DevKitC |
| 360° 连续旋转舵机 | 1 个 | SG90 360 版或其他 360 舵机 |
| 杜邦线 | 若干 | 公对母或母对母 |
| 5V 电源 | 1 个 | 建议使用外接电源 |

## 🔌 硬件接线

```
360°舵机     ESP32-S3
棕色 (GND)  →  GND
红色 (VCC)  →  5V (或 VIN)
橙色 (SIG)  →  GPIO40
```

> ⚠️ **注意**: 舵机工作电流较大，建议使用外接 5V 电源供电

## 💻 软件配置

### 1. 安装 Arduino IDE

- 下载并安装 [Arduino IDE](https://www.arduino.cc/en/software)
- 安装 ESP32 开发板支持 (版本 3.3.7 或更高)

### 2. 安装依赖库

在 Arduino IDE 中：
- 项目 → 加载库 → 管理库
- 搜索并安装 **ESP32Servo** (by Kevin Harrington)

### 3. 配置 WiFi

打开 `servo_controller/servo_controller.ino`，修改：

```cpp
const char* ssid = "YOUR_WIFI_SSID";        // 你的 WiFi 名称
const char* password = "YOUR_WIFI_PASSWORD"; // 你的 WiFi 密码
```

### 4. 上传程序

1. 选择开发板：**ESP32S3 Dev Module** 或 **Xiao ESP32S3**
2. 选择正确的 COM 端口
3. 上传代码 (首次上传需按住 BOOT 键)
4. 打开串口监视器 (115200 波特率)
5. 记录显示的 IP 地址

## 🌐 使用方法

### 访问 Web 界面

1. 在浏览器输入 ESP32-S3 的 IP 地址 (如 `http://192.168.1.100`)
2. 或通过 mDNS 访问：`http://esp32servo.local`

### 控制方式

**滑块控制**：
- 向左拖动：反转 (负值)
- 中间位置：停止
- 向右拖动：正转 (正值)

**档位按钮**：
- 🔴 **反转**：全速反转 (-100%)
- ⏹ **停止**：立即停止 (0%)
- 🟢 **慢速**：半速正转 (50%)
- 🚀 **快速**：全速正转 (100%)

**快捷按钮**：
- ← **反转**：全速反转
- ⏹ **停止**：立即停止
- **正转 →**：全速正转

## 🔧 代码说明

### 核心功能

```cpp
// 360 度舵机速度映射
if (speed == 0) {
    servoAngle = 90;  // 中位停止
} else if (speed > 0) {
    // 正转：90° 到 180° (速度越快，角度越大)
    servoAngle = map(speed, 0, 100, 90, 170);
} else {
    // 反转：90° 到 0° (速度越快，角度越小)
    servoAngle = map(abs(speed), 0, 100, 90, 10);
}
```

### Web 服务器路由

- `GET /` - 返回 Web 控制界面
- `GET /servo?speed=xxx` - 设置舵机速度
- `GET /status` - 获取当前状态
- `GET /speed?value=xxx` - 更新速度

## ⚠️ 注意事项

1. **供电**：360 舵机启动电流较大，建议使用外接 5V/2A 电源
2. **散热**：长时间运行注意舵机和 ESP32 的散热
3. **堵转**：避免长时间堵转，会损坏舵机齿轮
4. **调试**：不同品牌 360 舵机的中位点可能略有偏差，可通过代码微调

## 🛠️ 常见问题

### Q: 舵机抖动严重？
**A**: 供电不足，请使用外接 5V 电源并确保 GND 共地。

### Q: 无法连接 WiFi？
**A**: 检查 SSID 和密码是否正确，确保 ESP32-S3 在路由器覆盖范围内。

### Q: 舵机不停旋转？
**A**: 这是 360 舵机的正常特性，它无法像 180 舵机那样定位角度。

### Q: 中位不停止？
**A**: 不同舵机中位点可能有偏差，调整代码中的 `servoAngle = 90` 值 (尝试 88-92)。
