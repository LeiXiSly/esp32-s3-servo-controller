# ESP32-S3 360° 舵机控制器

基于 ESP32-S3 的 360 度连续旋转舵机 WiFi 控制器，支持通过 Web 界面实时控制舵机转速和方向。

## 🌟 特性

- ✅ **360° 连续旋转** - 支持正转、反转、停止控制
- ✅ **Web 界面控制** - 手机/电脑浏览器直接访问
- ✅ **无级调速** - -100% 到 100% 精确速度控制
- ✅ **实时反馈** - 状态实时显示
- ✅ **可视化动画** - 舵机旋转状态实时显示

## 📦 所需材料

| 物品 | 数量 |

## 🔌 硬件接线

```
360°舵机     ESP32-S3
棕色 (GND)  →  GND
红色 (VCC)  →  5V
橙色 (SIG)  →  GPIO40
```

## 💻 使用说明

### 1. 配置 WiFi

打开 `servo_controller/servo_controller.ino`，修改：

```cpp
const char* ssid = "YOUR_WIFI_SSID";
const char* password = "YOUR_WIFI_PASSWORD";
```

### 2. 上传代码

1. 使用 Arduino IDE 打开 `servo_controller/servo_controller.ino`
2. 选择开发板：**ESP32S3 Dev Module**
3. 选择 COM 端口
4. 点击上传

### 3. 访问控制界面

1. 打开串口监视器（115200 波特率）
2. 记录 IP 地址（如 `192.168.1.100`）
3. 浏览器访问：`http://192.168.1.100`

### 4. 控制舵机

- **滑块**：左右拖动调节速度
- **按钮**：反转/停止/慢速/快速
- **快捷**：一键正转/反转/停止

## 🛠️ 常见问题

**Q: 舵机抖动？**  
A: 供电不足，请使用外接 5V 电源。

**Q: 无法连接 WiFi？**  
A: 检查 SSID 和密码，确保 ESP32-S3 在路由器覆盖范围内。

**Q: 舵机不停旋转？**  
A: 这是 360 舵机的正常特性，无法定位角度。
